# Technical Test Junior FE 

Backend API untuk aplikasi Technical Test Junior FE .

## Persyaratan Sistem

- Node.js (versi 16 atau lebih tinggi)
- npm atau yarn

##  Langkah-langkah Instalasi Dependensi

**Clone repository** (jika belum):
   ```bash
   git clone https://github.com/username/FE-PT-SGT.git

   cd technical-test-fe
   ```
## Instalasi

### Install dependencies
Gunakan salah satu perintah berikut:

```bash
# Menggunakan yarn (disarankan)
yarn install

# Atau menggunakan npm
npm install
```

## Menjalankan Aplikasi

### Mode Development 
```bash
# Menggunakan yarn
yarn dev

# Atau menggunakan npm
npm run dev
```

Server akan berjalan di `http://localhost:3000`
